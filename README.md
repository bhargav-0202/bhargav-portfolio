# Bhargav Portfolio

This repository contains my personal portfolio website built to showcase my work as a Data Engineer.

## Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS

## Features
- Modern, responsive UI
- Animated sections and transitions
- Skills, projects, certifications, and education sections
- Contact form with interactive UI
